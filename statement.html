I confirm that the work submitted in this assignment is my own. Where I have drawn on third‑party material, libraries or open‑source code, the source is acknowledged on the References page and inline in the source code where appropriate.

My own work

  • All HTML, CSS and JavaScript in this project (excluding clearly identified third‑party libraries listed below).
  • The MVC architecture and the front‑end / back‑end split.
  • The custom GLSL fresnel rim‑light shader (js/shaders/fresnel.vert, js/shaders/fresnel.frag).
  • The PHP / Slim REST API and SQLite schema (api/index.php).
